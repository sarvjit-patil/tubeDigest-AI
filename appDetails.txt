# Project Overview
TubeDigest AI is a web application that elevates the YouTube experience by providing professional AI-driven summaries and sentiment analytics for YouTube videos. 

# Core Features
1. YouTube URL Parsing: Accepts a standard YouTube URL and extracts the video ID and thumbnail.
2. Transcript Extraction: Fetches the video transcript automatically using `youtube-transcript-api`.
3. AI Summarization: Processes the video transcript using the Groq API (Llama 3.1) to generate key highlights and a comprehensive summary. Handles large transcripts by splitting text into chunks and managing rate limits automatically.
4. Sentiment Analytics: Analyzes the overall mood of the video content, calculating a Sentiment Score between 0 and 100.
5. Interactive Visualizations: Displays the audience mood using a dynamic Plotly gauge chart.

# Technology Stack
- Frontend UI: Streamlit (Python web framework)
- UI Styling: Custom CSS (`styles/custom_css.py`)
- AI Model: Groq API (Llama 3.1 8B Instant)
- Data Visualization: Plotly (`plotly.graph_objects`)
- YouTube Integration: `youtube-transcript-api`, `yt-dlp`
- Text Processing: `langchain-text-splitters`

# Project Structure
- `app.py`: The main entry point for the Streamlit application. Contains the UI layout, routing, and main application logic.
- `requirements.txt`: Lists all the necessary Python dependencies for the project.
- `utils/youtube_helper.py`: Contains utility functions to interact with YouTube (e.g., `extract_video_id`, `get_video_transcript`, `get_video_thumbnail_url`).
- `utils/ai_helper.py`: Contains AI-related logic, specifically the `get_summarization_and_sentiment` function that communicates with the Groq API and handles rate limits.
- `styles/custom_css.py`: Contains the `apply_custom_css` and `card_container` functions to apply custom, sleek styling to the Streamlit app.
- `package.json`: Contains scripts (like `npm run dev`) mapped to run the Streamlit application.
- `.devcontainer/`: Configuration for Development Containers.

# How to Run
1. Ensure all dependencies from `requirements.txt` are installed.
2. Run the application using the command: `streamlit run app.py` (or `npm run dev` if configured).
3. Enter a valid Groq API key in the sidebar of the application to enable summarization features.
